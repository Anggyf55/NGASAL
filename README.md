## Setup

### Requirements
- Python 3.8+
- Flask
- Flask-CORS
- requests

### Installation

1. Clone repository:
```bash
git clone https://github.com/Anggyf55/NGASAL.git
cd NGASAL
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run server:
```bash
python server.py
```

4. Buka browser: `http://localhost:5000`
